Pengaturan

Buka Kunci Tanggal

Tanggal yang sudah di-export terkunci otomatis. Pilih tanggal untuk membuka kuncinya agar bisa diedit kembali.

Pilih tanggal untuk melihat statusnya.

Ganti PIN

PIN digunakan untuk mengunci panel Pengaturan ini.

Nomor WhatsApp Admin

Nomor ini digunakan untuk tombol kirim laporan di halaman Pencocokan. Format: angka saja tanpa tanda + atau strip, diawali kode negara.
Contoh: 6281234567890

Nama Outlet

Wajib diisi sesuai lokasi outlet ini. Otomatis ditambahkan ke nama file Excel, file backup, dan teks WhatsApp — supaya pusat tahu laporan ini dari outlet mana.

Backup Data

Unduh seluruh data transaksi semua tanggal sebagai file .json. Simpan sebagai cadangan — bisa digunakan untuk restore jika data hilang. Nama file bisa disesuaikan sebelum diunduh.

Restore Data

Muat data dari file backup .json. Data di-restore akan digabung dengan data yang sudah ada.

Hapus Semua Data

Hapus seluruh data dari semua tanggal. Lakukan backup terlebih dahulu. Tidak bisa dibatalkan.
